# How To Use
## Step 1 - clone this repo
```sh
$ cd ~/
$ git clone https://github.com/yhoshino11/.vimrc_rMBP15.git
```

## Step 2 - create symbolic link
```sh
$ cd ~/
$ ln -s .vimrc_rMBP15/.vimrc .vimrc
```
